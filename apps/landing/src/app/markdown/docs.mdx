React Beauty Flags

# Documentation

React Beauty Flags is a simple React component that renders a flag of a country. It is a simple and easy way to display a flag in your React application.

## Installation

To install React Beauty Flags, you can use npm, pnmp, or yarn.

```bash
pnpm install react-beauty-flags
```

## Usage

To use React Beauty Flags in your application, you need to import the component and pass the `countrycode` as a prop. Here is an example:

```jsx
import Flag from "react-beauty-flags"

const App = () => {
  return <Flag code="US" />
}
```
